# 📅 Just Today

一款簡潔直覺的每日待辦事項管理系統，支援任務新增、時間管理、標籤分類與清單預覽，幫助你專注完成每一件重要的小事。

---

## ✨ 功能特色

- ✅ 任務表單：可填寫標題、描述、開始/結束時間、標籤
- 🔎 任務預覽：即時顯示已建立任務清單
- 🏷️ 標籤分類：支援「學校 / 私人 / 工作」與自訂標籤
- 🎨 美觀介面：黑白簡約風搭配陰影圓角設計
- 📌 頁面導覽列：含「今日事項 / 代辦事項 / 番茄鐘 / 摸魚 / 關於我們」

---

## 🔧 使用技術

- React + Vite
- Tailwind CSS + DaisyUI
- React Router v6
- React Helmet（SEO / 標題控制）
- Git / GitHub（版本控制）
- 可選 Redux（狀態管理，視專案擴充性）
